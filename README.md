Secure File Sharing System
This is a simple web application to upload, encrypt, and download files using AES encryption.
It is built using Python Flask as part of my internship Task 3.

Features
Upload any file from your computer
Encrypt the file using AES
Download encrypted or decrypted version
How to Run
Make sure Python is installed on your computer.

Install required packages: pip install -r requirements.txt

Run the Flask app: python application.py

Open your browser and go to: http://127.0.0.1:5000/

Upload a file and test the encrypted and decrypted download links.

Tools & Libraries
Python 3
Flask
PyCryptodome
